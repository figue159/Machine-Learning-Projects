# Customer Churn Prediction

Este proyecto implementa un modelo de **Machine Learning en Python** para predecir la probabilidad de que un cliente abandone un servicio (*churn*). Es un problema clásico en analítica de datos y muy relevante en entornos empresariales, ya que permite a las compañías anticipar pérdidas de clientes y diseñar estrategias de retención.

---

##  Objetivos del Proyecto
- Realizar un **tratamiento de datos** completo: limpieza, codificación de variables categóricas y normalización.  
- Implementar y comparar **algoritmos de clasificación** (Regresión Logística y Random Forest).  
- Evaluar el desempeño del modelo con **métricas profesionales**: accuracy, precision, recall, f1-score y AUC.  
- Visualizar resultados con **gráficos claros y ejecutivos**: matriz de confusión, curva ROC y análisis de importancia de variables.  

---

## 🛠️ Tecnologías Utilizadas
- **Python 3.9+**
- **Pandas** – manipulación y análisis de datos  
- **NumPy** – operaciones numéricas  
- **Scikit-learn** – algoritmos de machine learning y métricas  
- **Matplotlib & Seaborn** – visualización de datos  
- **XGBoost (opcional)** – modelo avanzado de boosting
